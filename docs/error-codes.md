# 错误码字典

> 全平台只能返回此字典内的 code。新增 code **必须**在本文件登记。

格式：`{ "code": <int>, "message": <str>, "data": null, "trace_id": <uuid> }`。

---

## 段位约定

| 段位     | 业务域           |
| -------- | ---------------- |
| 1xxxx    | 通用             |
| 2xxxx    | 用户 / 权限      |
| 3xxxx    | 事件上报         |
| 4xxxx    | 预警 / 知识库    |
| 5xxxx    | 安全教育         |
| 6xxxx    | 审计 / 司法协助  |
| 9xxxx    | 外部依赖         |

---

## 1xxxx 通用

| code  | message                          | HTTP | 触发场景                                   |
| ----- | -------------------------------- | ---- | ------------------------------------------ |
| 10000 | 服务器内部错误                   | 500  | 未捕获异常                                 |
| 10001 | 请求参数校验失败                 | 422  | Pydantic 校验未通过                        |
| 10002 | 资源不存在                       | 404  | get/path 找不到                            |
| 10003 | 资源冲突                         | 409  | 创建已存在记录                             |
| 10004 | 请求过于频繁，请稍后再试         | 429  | 限流                                       |
| 10005 | Idempotency-Key 已被使用且参数不一致 | 409 | 同 key 不同 body                          |
| 10010 | CSRF: 缺少 X-Requested-With 头   | 400  | 写接口无 CSRF header                       |

## 2xxxx 用户 / 权限

| code  | message                              | HTTP | 触发场景                              |
| ----- | ------------------------------------ | ---- | ------------------------------------- |
| 20001 | 请先登录                             | 401  | session 缺失/过期                      |
| 20002 | 权限不足                             | 403  | RBAC 拦截                              |
| 20003 | 账号已被停用                         | 403  | status=2                              |
| 20004 | 会话已被吊销                         | 401  | revoke_all_for_user 后                 |
| 20005 | CAS 票据无效或已过期                 | 401  | CAS validate 失败                      |
| 20006 | CAS 票据已被使用（重放保护）         | 401  | TicketDedup 命中                       |
| 20007 | 回跳地址不在白名单内                 | 400  | OpenRedirectBlocked                    |

## 6xxxx 审计 / 司法协助

## 3xxxx 事件上报域

| code  | message                          | HTTP | 触发场景                                              |
| ----- | -------------------------------- | ---- | ----------------------------------------------------- |

## 4xxxx 预警 / 知识库

| code  | message                          | HTTP | 触发场景                                                  |
| ----- | -------------------------------- | ---- | --------------------------------------------------------- |
| 40001 | 预警业务规则校验失败             | 400  | 预警通用兜底                                              |
| 40002 | 预警不存在                       | 404  | 预警 ID 找不到                                            |
| 40003 | 预警已下线                       | 409  | 在已下线预警上做发布 / 追加                               |
| 40004 | 预警参数非法                     | 422  | level / scope 不合法 / dept 列表为空                      |
| 40010 | 知识库条目业务规则校验失败       | 400  | 知识库条目通用兜底                                        |
| 40011 | 知识库条目不存在                 | 404  | 条目 ID 找不到                                            |
| 40012 | 非法的条目状态转换               | 409  | 例如已发布 → 重新提交                                     |
| 40013 | 条目搜索参数非法                 | 422  | 关键词为空 / 分页越界 / 过滤字段不合法                    |

## 5xxxx 安全测验 / 题库（UC-05 / UC-09）

| code  | message                          | HTTP | 触发场景                                  |
| ----- | -------------------------------- | ---- | ----------------------------------------- |
| 50001 | 测验业务规则校验失败             | 400  | 测验通用兜底（QuizError）                 |
| 50002 | 测验不存在                       | 404  | quiz_id 不存在（QuizNotFound）            |
| 50003 | 题目不存在                       | 404  | question_id 不存在（QuestionNotFound）    |
| 50004 | 测验已结束或已撤回               | 409  | 测验不可参与/不可开始（QuizClosed）       |
| 50005 | 已提交答案，不可重复提交         | 409  | attempt 已提交，重复提交（QuizAlreadySubmitted） |
| 50006 | 未在本次测验的参与范围内         | 403  | 非目标用户参与指定测验（QuizNotEligible） |
| 50007 | 题库可用题目数不足，无法生成测验 | 422  | 题库可用题目不足（QuizQuestionBankExhausted） |
| 50008 | 测验参数非法                     | 422  | 分页/筛选/输入参数不合法（QuizInvalidParam） |
